## 📖 Descripción
Una aplicación web autónoma, de **un solo archivo**, que consolida los mejores principios de `TokenDensity`, `PromptElite`, `PromptCraft`, `Kimi Elite` y `ArchitectPrompts`. Diseñada para ingenieros, arquitectos de software y desarrolladores que exigen **máxima densidad semántica, cero desperdicio de tokens y generación de código Android compilable sin IDE**.

✨ **Filosofía:** *No le pidas a la IA que adivine. Dile exactamente qué hacer, cómo hacerlo, y qué NO debe hacer.*

## 🔧 Personalización & Despliegue
Este proyecto es una **Single-File Application**. Para modificarlo o desplegarlo:

1. **Editar:** Abre `PromptArchitect_Ultra.html` en VS Code. Todo el HTML, CSS y JS está inline.
2. **Añadir Prompts:** Busca el objeto `androidPrompts` en la etiqueta `<script>` y añade nuevas entradas siguiendo el formato `id: "prompt_texto"`.
3. **Cambiar Tema:** Modifica las clases de Tailwind o ajusta las variables CSS personalizadas en `<style>`.
4. **Desplegar:** Sube el archivo a cualquier hosting estático:
   - 🐙 GitHub Pages
   - ▲ Vercel / Netlify / Cloudflare Pages
   - 🌐 Servidor Nginx/Apache tradicional
